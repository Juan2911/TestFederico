# TestFederico

Se recibirá como parámetro un objeto Personas, el cual será una lista de objeto Persona, que contiene nombre y edad

se solicita que se ordene dicha lista de menor a mayor por edad y devolver una lista de personas con el resultado

datos de ejemplo:
{
  "personas": [
    {
      "nombre": "roberto",
      "edad": 40
    },
    {
      "nombre": "diego",
      "edad": 35
    },
    {
      "nombre": "david",
      "edad": 15
    }
  ]
}

resultado esperado

[
    {
        "nombre": "david",
        "edad": 15
    },
    {
        "nombre": "diego",
        "edad": 35
    },
    {
        "nombre": "roberto",
        "edad": 40
    }
]
